# Day 1 촬영 체크리스트 — 제작기·설치 (2편용)

> **목표**: 2편 "제작기 — 사장님 × 포토네모"에 필요한 모든 소스 촬영  
> **예상 소요**: 4~5시간 (오전 2h + 오후 2~3h)

---

## 촬영 전 준비물

### 장비
- [ ] 스마트폰 2대 (1대: 메인 촬영, 1대: 화면 녹화)
- [ ] 삼각대 또는 스마트폰 거치대
- [ ] 마이크 (사장님 인터뷰용, 있으면)
- [ ] 보조 배터리

### 소프트웨어/자료
- [ ] 키오스크 PC — 앱 실행 가능 상태
- [ ] 시행착오 스크린샷 3~4장 (없으면 당일 재촬영)
  - 너무 어려 보이는 결과
  - 너무 늙어 보이는 결과
  - 소품 왜곡 (칼 너무 큼 등)
  - 최종 성공 결과
- [ ] 노트/화이트보드 + 마커 (기획 장면용)
- [ ] `simmar.md` 참고 — 실제 피드백 사례

### 인원
- [ ] 키즈카페 사장님 (주인공)
- [ ] 포토네모 담당자 1명
- [ ] 촬영 담당 1명

---

## 오전 세션 (10:00~12:00) — 기획·협업·시행착오

### 컷 1: 훅용 실패 결과 (0~2초용)
| 항목 | 내용 |
|---|---|
| **화면** | 너무 어리게/늙게 나온 AI 결과 사진 → 사장님 당황 표정 |
| **촬영** | 태블릿/모니터에 결과 띄우고 사장님 반응 클로즈업 |
| **대사** | 없음 (자막만: `7살 아이가` `65살 할머니가 됐습니다`) |
| **체크** | [ ] 실패 결과 화면 [ ] 사장님 당황 표정 |

### 컷 2: 사장님 독백 (2~8초용)
| 항목 | 내용 |
|---|---|
| **장소** | 카페 한쪽, 배경 정리된 곳 |
| **구도** | 가슴 위 클로즈업, 정면 또는 45도 |
| **대사** | "처음 테스트할 때 진짜 당황했어요. 우리 아이들 얼굴이… 완전 다른 사람이 되더라고요." |
| **톤** | 진지하지만 따뜻하게 |
| **체크** | [ ] 테이크 2회 이상 [ ] 배경 소음 최소 |

### 컷 3: 기획 장면 B-roll (8~15초용)
| 항목 | 내용 |
|---|---|
| **화면** | 노트/화이트보드에 "아이들 꿈 키우기" 스케치 |
| **추가** | 직업 아이콘(의사🩺 소방관🚒 유튜버▶️) 그리기 장면 |
| **나레이션** | (후반 VO) "그래도 포기 못 했어요. 키즈카페 사장이니까…" |
| **체크** | [ ] 손글씨 스케치 [ ] 직업 아이콘 클로즈업 |

### 컷 4: 포토네모 미팅 (15~22초용)
| 항목 | 내용 |
|---|---|
| **장소** | 카페 테이블 또는 키오스크 PC 앞 |
| **화면** | 노트북에 앱 화면(직업 선택 UI), 두 사람 대화 |
| **대사 (사장님)** | "그래서 기술 파트너 **포토네모**한테 부탁했어요." |
| **대사 (포토네모)** | "직업 18개, 나이대별로 얼굴은 그대로 두고 나이만 바꾸는 거 — 생각보다 어렵습니다." |
| **추가** | "이 직업도 넣어주세요" 하며 화면 가리키는 장면 |
| **체크** | [ ] 앱 화면 선명 [ ] 두 사람 대화 [ ] 포토네모 10초 이내 |

### 컷 5: 시행착오 몽타주 소스 (22~30초용)

4컷 빠르게 촬영 또는 스크린샷 확보:

| 컷 | 화면 | 자막 (편집 시) | 체크 |
|---|---|---|---|
| A | 7살 아이 → 30대 의사인데 얼굴 너무 어려 보임 | `1차: 너무 어려요 😅` | [ ] |
| B | 같은 아이 → 40대처럼 너무 늙어 보임 | `2차: 이번엔 너무 늙었어요` | [ ] |
| C | 요리사인데 칼이 너무 큼 | `소품도 문제…` | [ ] |
| D | 최종 — 자연스러운 미래 모습 | `수십 번 조율 끝` | [ ] |

> **팁**: 과거 테스트 스크린샷이 있으면 재촬영 없이 활용 (`simmar.md` 피드백 참고)

### 컷 7: 최종 결과 Before/After (40~50초용)
| 항목 | 내용 |
|---|---|

| **체크** | [ ] 결과 3종 이상 [ ] 얼굴 닮음이 잘 보이는 샷 |

### 컷 8: 엔딩 (50~60초용)
| 항목 | 내용 |
|---|---|
| **화면** | 사장님 + 포토네모 한 컷 (선택) |
| **대사 (사장님)** | "방학이면 아이들한테 '놀 곳'만 주는 게 아니라, '꿈 키우는 시간'을 주고 싶었어요." |
