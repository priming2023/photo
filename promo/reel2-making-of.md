## 대본 (전체)

### [0~2초 — 훅]
- **화면**: 너무 어리게/늙게 나온 AI 결과 → 사장님 당황 클로즈업
- **자막** (크게, 2줄):
  - `7살 아이가`
  - `65살 할머니가 됐습니다 (AI 실수)`
- **사운드**: 짧은 충격음 또는 "띠용—"

### [2~8초 — 사장님]
- **대사**: "처음 테스트할 때 진짜 당황했어요. 우리 아이들 얼굴이… 완전 다른 사람이 되더라고요."
- **화면**: 사장님 정면, 진지한 톤

### [8~15초 — 기획 B-roll + VO]
- **화면**: 노트에 "아이들 꿈 키우기" 스케치, 직업 아이콘
- **나레이션 (VO)**: "그래도 포기 못 했어요. 키즈카페 사장이니까, 아이들이 매일 오는 곳에서 '나도 할 수 있어'를 느꼈으면 했거든요."

### [15~22초 — 포토네모 협업]
- **사장님**: "그래서 기술 파트너 **포토네모**한테 부탁했어요."
- **포토네모** (10초 이내): "직업 18개, 나이대별로 얼굴은 그대로 두고 나이만 바꾸는 거 — 생각보다 어렵습니다."
- **자막**: `포토네모 × 월드킹` `직접 기획 · 직접 개발`
- **B-roll**: 노트북 앱 화면, "이 직업도 넣어주세요" 대화

### [22~30초 — 시행착오 몽타주]

각 컷 1.5~2초, 빠른 전환:

| 컷 | 화면 | 자막 | 전환 |
|---|---|---|---|
| A | 7살 → 30대 의사인데 얼굴 너무 어려움 | `1차: 너무 어려요 😅` | 슬라이드 |
| B | 같은 아이 → 40대처럼 너무 늙음 | `2차: 이번엔 너무 늙었어요` | 슬라이드 |
| C | 요리사, 칼이 너무 큼 | `소품도 문제…` | 슬라이드 |
| D | 최종 자연스러운 결과 | `수십 번 조율 끝` | 페이드 |

- **나레이션 (VO)**: "너무 어렵게, 너무 젊게, 소품은 너무 크게… 한 번에 되는 게 없더라고요."

### [30~40초 — 설치 현장]
- **나레이션 (VO)**: "그래도 포토네모랑 같이 프린터도 연결하고, 키오스크도 세팅하고…"
- **B-roll 순서**:
  1. 키오스크 PC 세팅
  2. windows-build.bat 또는 exe 설치
  3. .env에 PRINTER_NAME 입력
  4. 테스트 영수증 출력

### [40~50초 — 최종 결과]
- **사장님**: "이제는 아이 얼굴은 그대로, 꿈의 직업복만 입혀줘요."
- **B-roll**: 의사 / 소방관 / 유튜버 결과 3장 (2초씩)

### [50~60초 — 엔딩]
- **사장님**: "방학이면 아이들한테 '놀 곳'만 주는 게 아니라, '꿈 키우는 시간'을 주고 싶었어요."
- **자막**:
  - `7월 방학`
  - `월드킹 당진서산점`
  - `꿈을 이룬 내모습 사진관`
  - `아이들 반응은 3편 🎬`

## 편집 체크리스트

### 훅 (0~2초)
- [ ] 실패 결과가 **첫 화면** — 스크롤 멈춤 유도
- [ ] 자막 크고 읽기 쉽게 (충격적 메시지)
- [ ] 사장님 당황 표정 0.5초 이상

### 시행착오 몽타주
- [ ] 4컷 각 1.5~2초 (총 6~8초)
- [ ] 전환: 슬라이드 또는 빠른 컷 (0.2초)
- [ ] BGM: 잔잔한 피아노, 긴장→해소 아크

### 포토네모
- [ ] 등장 10초 이내
- [ ] 기술 파트너로 신뢰감 (과하지 않게)

### BGM
- 2편: 잔잔한 피아노 또는 어쿠스틱
- 몽타주 구간: 템포 약간 업
