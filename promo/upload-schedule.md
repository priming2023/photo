### 공통 (매 업로드 시)
- [ ] 비율 9:16 확인
- [ ] 자막 소리 꺼도 읽히는지 확인
- [ ] 썸네일: 가장 임팩트 있는 프레임 선택
- [ ] 위치 태그: 당진, 충남 당진시
- [ ] 계정 태그: @포토네모 (해당 시)

- [ ] 스토리에도 공유 (링크 유도)

- [ ] 파일: `reel2_making_of.mp4`

- [ ] 캡션 붙여넣기 (아래)

### 3편 업로드 (오픈 당일)
- [ ] 파일: `reel3_launch_reaction.mp4`

- [ ] 캡션 붙여넣기 (아래)

## 스토리 활용

| 시점 | 스토리 내용 |
|---|---|
| 1편 게시일 | 릴스 공유 + "곧 오픈해요" 투표/질문 스티커 |
| 2편 게시일 | 제작 비하인드 짧은 클립 + "어떤 직업 고를래?" |
| 3편 게시일 | 오픈 안내 + 위치 스티커 + "DM 주세요" |

## 성과 확인 (업로드 후)

| 지표 | 확인 시점 | 목표 |
|---|---|---|
| 조회수 | 24시간 후 | 1편 > 2편 > 3편 (훅 효과) |
| 저장 수 | 48시간 후 | 1편에서 높을수록 좋음 |
| DM 문의 | 3편 이후 | 방문 문의 증가 |
| 완시청률 | 인사이트 | 1편 50%+, 2편 40%+ |
