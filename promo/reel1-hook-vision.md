## 대본 (전체)

### [0~1.5초 — 화면만]
- **화면**: 아이 손이 키오스크에서 의사/소방관 버튼 누름
- **사운드**: 완전 무음 0.5초 → **띵!** 효과음
- **자막**: `방학.` (0.3초) → `여기서 꿈이 자란다.` (0.5초)

### [1.5~6초 — 빠른 컷]
- **화면**: 셔터 → 로딩 스피너 → 결과 사진 공개 (각 0.5~1초)
- **사운드**: 카메라 셔터음 → 후우웅(변환) → 짜잔(결과)
- **자막**: `직업 고르기` → `찰칵` → `미래의 나`

### [6~12초 — 아이 반응]
- **화면**: 아이가 결과 사진 보며 입 벌리고 웃음, 부모 뒤에서 미소
- **사운드**: 아이 "우와~!" 또는 "엄마!"
- **자막**: `"엄마 나 의사 됐어!"`

### [12~18초 — 사장님]
- **화면**: 사장님 가슴 위 클로즈업, 카페 배경
- **대사**: "아이들이 '나 커서 뭐가 될까?' 물을 때마다… 그냥 놀아주는 것만으로는 부족하다고 느꼈어요."
- **톤**: 부드럽고 진심

### [18~22초 — 프린터]
- **화면**: 80mm 영수증이 프린터에서 천천히 나옴 (클로즈업)
- **사운드**: 프린터 모터 소리 (ASMR 느낌, BGM 낮춤)
- **자막**: `영수증으로 받고` → `QR로 폰에 저장`

### [22~25초 — 엔딩]
- **화면**: 로고/텍스트 카드 또는 카페 전경
- **자막**: 
  - `월드킹 당진서산점`
  - `꿈을 이룬 내모습 사진관`
  - `7월 방학 오픈`
  - `제작 과정은 2편 👇`

## 편집 체크리스트

### 비주얼 훅 (필수)
- [ ] **0~1.5초 무음** — 말 없이 강한 화면만
- [ ] 첫 프레임이 스크롤을 멈출 만큼 시선 끄는지 확인
- [ ] 자막 2~4단어씩 끊기 (소리 꺼도 이해 가능)

### 사운드
- [ ] 0~1.5초: 무음 또는 극히 낮은 앰비언트
- [ ] 1.5초: 띵! 또는 직업 선택 효과음
- [ ] 3~6초: BGM 진입 (밝고 희망적인 톤)
- [ ] 18~22초: 프린터 소리 강조 (BGM -6dB)

### 자막 스타일
- 위치: 화면 중앙 하단
- 폰트: 굵은 고딕, 흰색 + 검정 외곽선
- 길이: 한 줄 6자 이내 권장

### BGM
- 1편: 무음 시작 → 효과음 → 밝은 어쿠스틱/피아노 (저작권 프리)
- 볼륨: 나레이션 시 -12dB, 효과음 시 BGM 뮤트
